# markdown-editor

Write Markdown on the left, see the rendered output on the right.

## Features

- Split-pane with live preview
- CommonMark + GFM (tables, strikethrough, task lists, fenced code blocks)
- Toolbar: bold, italic, heading, link, image, code, blockquote, list
- Word count and character count
- Export as HTML or download as .md
- Auto-save every 2 seconds (localStorage)
- Dark and light themes

## Run

```bash
npm install && npm run dev
```
