# Markdown Editor

Live Markdown editor with a split-pane preview — write on the left, see the rendered output on the right instantly.

## Features

- Split-pane editor and live preview
- Full CommonMark support + GFM (tables, strikethrough, task lists)
- Toolbar for common formatting (bold, italic, heading, link, image, code block)
- Word and character count
- Export as HTML or download as `.md` file
- Auto-save to localStorage
- Dark and light themes
